📝 Project Description:
 
            

            Amazon Clone (Front-End Only)

            This project is a front-end replica of Amazon's homepage using only HTML and CSS.
            It focuses on structuring and styling key UI components to closely resemble the 
            original design of Amazon India’s website. The goal was to practice and demonstrate skills 
            in layout design, flexbox, image handling, and responsive styling principles.


            
✅ Key Features:


           
 Navigation Bar (.navbar):



           Includes a logo, delivery address section, and placeholders for account/cart info.
           Built using Flexbox for proper alignment and spacing.
           Font Awesome icons (like the location pin) are integrated.


            
Hero Section (.hero_section):
 

            Large promotional banner area with a centered call-to-action message.
            Uses a background image to mimic Amazon’s main carousel area.

            
Search Bar (.nav-search):
 


            Functional layout of a search box with a category dropdown and search icon.
            Responsive width with hover effects.


            
Second Navigation Panel (.pannel-2):



            Displays quick access links such as "All" and "Today's Deals".
            Styled with text alignment and bold white text on a dark background.

            
Product Boxes (.shop-section):



            A row of four boxes each representing a category (box1, box2, box3, box4).
            Uses background images and padding for spacing and design.
            Implemented with flex for even distribution across the page.

            
🎯 Technologies Used:


            HTML5
            CSS3 (including Flexbox)
            Font Awesome (via CDN)


            
📱 Responsive Design:


            The  tag is used to ensure proper scaling on mobile devices. 
            Some elements are relatively positioned for alignment, though full responsiveness can be 
            improved in future versions.
